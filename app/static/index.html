A privacy-first AI assistant that runs locally on your device. Think JARVIS from Iron Man, but real and respecting your data.

Core Features

  • Voice & Vision input (OCR, screen awareness)
  • Productivity tools (research, documents, notes)
  • Developer assistant (coding help, screen reading)
  • Accessibility features (helping visually impaired users)
  • Multi-platform (Desktop, Web, Mobile)

Tech Stack
